美团笔试-最长不下降子序列

发表于 2020-03-20 | 分类于 -笔试

最长上升子序列

介绍

最长不下降子序列，最长上升子序列，最长下降子序列等等，都类似。

最长不下降子序列即所给序列的子序列中满足不下降的最大长度

解题历程

em, 开始不懂什么是最长不下降子序列，就以为是连续的，然后就悲剧了。

这题要用bp算法，维持一个dp数组，数组中元素dp[i]表示长度为i的不下降子序列的最后一位中的最小值。从所给序列第一个元素开始，不断更新dp即可。
比如dp[4]=5,说明前面长度为4的子序列最后一位的最小值为5，这个时候下一位元素为6，直接添加dp[5]=6即可。如果下一位是3，我们需要在dp数组中找到比3大的元素，然后更新它的值为3，比如dp[3]=4,dp[2]=2,以4结尾的那个子串是也可以以3结尾，后面的元素范围比之前大一些。

代码

def ans(L):
    dp1 = [L[0]]
    for i, item in enumerate(L[1:]):

        if dp1[-1]<=item:
            dp1.append(item)
            continue
        for j, j_item in enumerate(dp1):
            if j_item > item:
                break
        dp1[j] = item
        print(dp1)
    return len(dp1)

leetcode-二叉树的堂兄弟节点

发表于 2020-03-16 | 分类于 tags ， categories

二叉树的堂兄弟节点

题目描述

题目链接
在二叉树中，根节点位于深度 0 处，每个深度为 k 的节点的子节点位于深度 k+1 处。

如果二叉树的两个节点深度相同，但父节点不同，则它们是一对堂兄弟节点。

我们给出了具有唯一值的二叉树的根节点 root，以及树中两个不同节点的值 x 和 y。

只有与值 x 和 y 对应的节点是堂兄弟节点时，才返回 true。否则，返回 false。

class Solution(object):
def isCousins(self, root, x, y):
    parent = {}
    depth = {}
    def dfs(node, par = None):
        if node:
            depth[node.val] = 1 + depth[par.val] if par else 0
            parent[node.val] = par
            dfs(node.left, node)
            dfs(node.right, node)

    dfs(root)
    return depth[x] == depth[y] and parent[x] != parent[y]
